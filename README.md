# Week1_AI-Prompts
Prompt 1 (Role and Task)- you are a professional therapist and psychologist working with teenagers aged 13-18. create a mental health/wellbeing app.  
Prompt 2 (Context)- you are a professional therapist and psychologist working with teenagers aged 13-18. create a mental health/wellbeing app called 'MoodFit'. It is specifically designed for teenagers to cope with exam stress .  
Prompt 3 (Constraints)- The interface of the app should be easy to navigate. Since its for students dealing with exams, make sure there are no distracting features. It should include solutions and ways to cope with stress. Also include a SOS number in case students need to reach out to professional help.  
Prompt 4 (Output)- Present the output as a detailed explanation of all the features, why they're included and how are they helpful.
